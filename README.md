# mon_association

Plateforme de gestion pour l'association — squelette initial .

Contenu poussé:
- backend/: projet Spring Boot (squelette, entités, repositories, controllers)
- frontend/: instructions et structure de base Angular
- docker-compose.yml : Postgres + placeholders pour backend/frontend

Base de données choisie : PostgreSQL
Inscription des membres : créée uniquement par l'administrateur

Instructions rapides (voir README dans chaque dossier):
- Backend : mvn spring-boot:run (ou builder Docker)
- Frontend : utiliser Angular CLI (instructions dans frontend/README.md)
